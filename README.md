# OhMyForm API

## Description

[OhMyForm](https://github.com/ohmyforn) api backend

All calls to the api are through GraphQL, with the endpoint 
providing an introspectable schema at `GET /graphql`

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
